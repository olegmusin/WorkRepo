Welcome to Shifts Calendar!

The Sence

This app demonstrates usage of the modern web related technologies for developing highly responsive, lightweight, scalable cross-platform solutions.
Its simple logics includes composition of a working shifts for abstract projects.
Each projects must have a fixed number of workers to fulfill the particular working shift. Each shift has the date and a list of the workers who has been assigned to this shift.
Thus you can manage shifts calendar in a obvious manner.

Features

  •   Projects Briefcase
  •   Compose Shifts
  •   Manage Workers

Back

Backend based on self-hosted .NET Core WebAPI project with EntityFrameworkCore, mapping MSSQL database automatically seeded with sample data.

Front

Frontend uses Angular 2 framework with ReactiveJS extensions for asynchroniously fetching api data using Observables approach.
TypeScript was chosen as base language due to its strong typing. Frontend core is running inside backend host, what became possible of middleware forwarding 404 backend request to frontend endpoints.
Source files are available on GitHub public repo.
